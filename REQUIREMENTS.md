# 需求文档 - Web Tools 在线工具聚合站

## 一、身份证水印生成器 (优先级：最高)

### 功能需求
1. **图片上传**
   - 支持点击上传
   - 支持拖拽上传
   - 图片预览

2. **水印配置**
   - 用途说明（必填）- 默认值："仅供办理XX业务使用"
   - 当前日期（自动填充，可修改）
   - 自定义文字（可选）
   - 字体大小、颜色设置（可选，使用默认值）

3. **水印样式**
   - 右下角斜铺模式
   - 半透明白色文字
   - 45度倾斜

4. **导出功能**
   - 实时预览水印效果
   - 一键下载生成后的图片

### 技术实现
- 使用 HTML5 Canvas API
- 参考小程序版实现逻辑
- 纯前端运行，无服务器

### UI/UX要求
- 简洁易用的界面
- 响应式设计（支持移动端）
- 上传区域明显可见
- 预览区域清晰展示效果

---

## 二、JSON格式化工具 (优先级：高)

### 功能需求
1. **输入区域**
   - 支持粘贴JSON文本
   - 支持上传JSON文件

2. **格式化功能**
   - 格式化（美化）JSON
   - 压缩JSON（去除空格）
   - 语法高亮

3. **错误处理**
   - 检测JSON语法错误
   - 显示错误位置和原因

4. **辅助功能**
   - 复制结果
   - 下载结果
   - 清空输入

### 技术实现
- 使用 JSON.parse() 解析
- 使用 JSON.stringify() 格式化
- 纯前端运行

---

## 三、二维码生成器 (优先级：中)

### 功能需求
1. **输入配置**
   - 文本/链接输入框
   - 二维码尺寸选择（128px, 256px, 512px）
   - 颜色选择（前景色、背景色）

2. **生成与预览**
   - 实时生成二维码
   - 预览二维码效果

3. **导出功能**
   - 下载为PNG图片

### 技术实现
- 使用 qrcode.js 库（CDN引入）
- 或使用 Google Charts API
- 纯前端运行

---

## 四、Base64编解码工具 (优先级：中)

### 功能需求
1. **Base64编码**
   - 文字编码
   - 图片转Base64

2. **Base64解码**
   - Base64转文字
   - Base64转图片（显示预览）

3. **辅助功能**
   - 复制结果
   - 下载结果

### 技术实现
- 使用 btoa() / atob() API
- FileReader API 处理图片
- 纯前端运行

---

## 五、全局设计要求

### 首页
1. 工具列表展示（卡片式布局）
2. 每个工具显示：图标、名称、简短描述
3. 搜索功能（后续扩展）
4. 响应式布局

### 公共组件
1. 导航栏（返回首页）
2. 页脚（版权信息）
3. 统一的配色方案
4. 统一的按钮样式

### 技术规范
1. 语义化 HTML5
2. CSS3 样式
3. ES6+ JavaScript
4. 跨浏览器兼容性
5. 移动端适配

---

## 六、性能与优化

1. 首页加载速度 < 2秒
2. 工具页面加载速度 < 1秒
3. 图片压缩（身份证水印工具）
4. 无阻塞交互

---

## 七、安全与隐私

1. 所有数据在浏览器本地处理
2. 不上传用户数据到服务器
3. 图片不保存到服务器

---

## 八、测试计划

1. 功能测试（每个工具的核心功能）
2. 浏览器兼容性测试（Chrome, Firefox, Safari, Edge）
3. 移动端测试（iOS, Android）
4. 性能测试

---

## 九、部署要求

1. 静态文件托管（GitHub Pages / Vercel）
2. HTTPS 支持
3. 自定义域名（可选）
4. CDN 加速（可选）
